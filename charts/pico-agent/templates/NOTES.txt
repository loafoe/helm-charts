To configure Grafana Alertmanager to send webhooks:
1. Get the webhook secret (see above)
2. Configure your contact point with:
   - URL: http://{{ include "pico-agent.fullname" . }}.{{ .Release.Namespace }}.svc.cluster.local:{{ .Values.service.port }}/task
   - HTTP Method: POST
   - Include the X-Grafana-Alertmanager-Signature header

Example payload for PV resize:
{
  "type": "pv_resize",
  "payload": {
    "namespace": "default",
    "pvc_name": "my-pvc",
    "new_size": "20Gi"
  }
}
